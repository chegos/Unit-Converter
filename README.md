# 🔁 Unit Converter Web App

Um conversor de unidades interativo feito com [Flask/Python], permitindo conversões entre comprimento, peso e temperatura de forma rápida e intuitiva.

---

## 📌 Sobre o Projeto

Este é um aplicativo web que converte diferentes unidades de medida:  
comprimento (mm, cm, m, km, in, ft, yd, mi), peso (mg, g, kg, oz, lb) e temperatura (°C, °F, K).  
O usuário insere um valor, seleciona a unidade de origem e a de destino, e o sistema retorna o valor convertido.

O projeto foi desenvolvido para praticar lógica de programação, manipulação de formulários web e rotas no backend.

https://roadmap.sh/projects/unit-converter

---

## 🚀 Funcionalidades

- ✅ Entrada de valor numérico
- ✅ Seleção de unidade de origem
- ✅ Seleção de unidade de destino
- ✅ Exibição do valor convertido
- ✅ Páginas separadas para:
  - 📏 Comprimento
  - ⚖️ Peso
  - 🌡️ Temperatura
- ✅ Submissão do formulário via `POST` para a própria página

---

## 🛠️ Tecnologias Utilizadas

- Python 3 (Flask)
- HTML5
- Sem banco de dados
